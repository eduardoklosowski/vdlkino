=======
VDLKino
=======

Library to control the Arduino with:

- Ethernet

- Serial


Block
=====

Commands are executed by blocks, these blocks have the following information:

===========  =======
Information  Size
===========  =======
oper         1 byte
pin          1 byte
value        2 bytes
===========  =======


Commands Return
===============

Commands return 2 bytes.
